# Changelog

## [0.1.0] - 2026-02-14

Initial public research release.

### Added
- Core quantum runtime: `Complex`, `QuantumState`, `Entanglement`
- Quantum operators: `QuantumGates`, `Interference`
- Quantum control flow: `QuantumConditionals`, `QuantumLoops`
- AI runtime: `QuantumLLM` with verification-weighted measurement
- Week 6 tools: `QuantumVisualizer`, `QuantumDebugger`, local playground
- AI benchmark harness with reproducible seeded evaluation
- Live Ollama demo pack (5 scenarios)
- Documentation suite (API, tutorial, theory, eval, install, quickstart)

### Validation
- Unit/integration tests passing
- Accuracy benchmark report generation working
- Benchmark and demos runnable from npm scripts

### API Stability Note
- APIs are considered stable for `0.1.x` research usage.
- Backward-compatible enhancements expected for minor updates.
